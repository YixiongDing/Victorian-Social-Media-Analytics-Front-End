Hide the title on single pages and posts.

== Description ==

Hide page and post title plugin provides functionality to user to hide the page and post title.

= Major features of this plugin include =

 1. Hide the title of particular page.
 2. Hide the title of particular post.
 3. Hide the title of particular custom post(CPT).

== Installation ==

The plugin is simple to install:

 1. Download hide-page-and-post-title.zip
 2. Unzip
 3. Upload the 'hide-page-and-post-title' folder to your /wp-content/plugins directory
 4. Go to the plugin menu page and activate the plugin

== Frequently asked questions ==

= How to hide the title of page or a post? =

 1. Activate the plugin through the 'Plugins' menu in WordPress.
 2. Then Create New Post/Page Or you can use old one.
 3. Now go to edit screen for pages or posts and click on the Hide the title options.
 4. After you click on "Hide the title" just publish or update the page or post.

= What is the benefit of using this plugin? =

User can easily hide the title of posts, pages or the custom posts.
